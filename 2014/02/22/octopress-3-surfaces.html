Octopress 3 Surfaces

22 Feb 2014

In this blog post we're going to talk a bit about Octopress 3.

Octopress is a framework designed by Brandon Mathis for Jekyll, the blog aware static site generator powering Github Pages.

Octopress 3 is described as A Ruby Command-Line Interface shipped as a Gem and as such is a complete rewrite of the old Octopress that we all love. It's currently at 3.0.0.alpha6 and getting better with every release. It is in no way complete however and is missing support for themes and plugins for example. There is also some oddities (missing bits) with the Markdown portion of it.

Regardless once completed, Octopress 3 will be an pretty significant upgrade and improvement over Octopress 2.

Installation

Add this line to your application's Gemfile:

gem 'octopress'

And then execute:

$ bundle

Or install it yourself as:

$ gem install octopress

Usage

Generating a new site

To create a new scaffold of directories and files in a new directory named my_blog:

$ octopress new my_blog

Generating a new Post

To create a new post just run:

$ octopress new post "My Title"

Generating a new Page

$ octopress new page --path about/index.markdown
